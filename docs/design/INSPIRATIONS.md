## 1. 🟡 Linear — Gold Standard UX

**Website:** linear.app
**Co bierzemy:** ⭐⭐⭐⭐⭐

```
Linear Influence on Nexus
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  ✅ Command palette (Cmd+K) — global search + actions
  ✅ Keyboard-first navigation — j/k, /, shortcuts
  ✅ Issue/Agent card design — clean, dense, actionable
  ✅ Activity feed — real-time, grouped, timestamped
  ✅ Settings layout — left nav, right content, diff preview
  ✅ Motion — snappy spring animations, purposeful micro-interactions
  ✅ Color usage — muted backgrounds, one accent color
  ✅ Empty states — illustrated, actionable
  ✅ Speed — sub-100ms interactions, optimistic UI
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**Kluczowa lekcja:** *„Every interaction should feel instant."* Linear buduje zaufanie przez szybkość. Nexus musi być równie szybki — żadnych spinnerów dłuższych niż 200ms.
